# Loop APK Build Instructions

আপনার অ্যাপটি APK হিসেবে তৈরি করার জন্য আমি **Capacitor** সেটআপ করে দিয়েছি। নিচের ধাপগুলো অনুসরণ করুন:

### ১. প্রজেক্ট ডাউনলোড করুন
প্রথমে AI Studio থেকে আপনার প্রজেক্টের জিপ (ZIP) ফাইল ডাউনলোড করে আপনার কম্পিউটারে আনজিপ করুন।

### ২. প্রয়োজনীয় সফটওয়্যার
আপনার কম্পিউটারে নিচের সফটওয়্যারগুলো থাকতে হবে:
- **Node.js** (v18 বা তার বেশি)
- **Android Studio**

### ৩. কমান্ডগুলো রান করুন
আপনার প্রোজেক্ট ফোল্ডারে টার্মিনাল বা কমান্ড প্রম্পট ওপেন করে নিচের কমান্ডগুলো দিন:

```bash
# ডিপেন্ডেন্সি ইন্সটল করুন
npm install

# অ্যান্ড্রয়েড প্ল্যাটফর্ম যোগ করুন (প্রথমবার)
npx cap add android

# অ্যাপটি বিল্ড করুন এবং অ্যান্ড্রয়েড প্রজেক্ট সিঙ্ক করুন
npm run cap:sync
```

### ৪. অ্যান্ড্রয়েড স্টুডিও দিয়ে ওপেন করুন
নিচের কমান্ডটি দিলে অ্যান্ড্রয়েড স্টুডিও ওপেন হবে:
```bash
npm run cap:open
```

### ৫. APK তৈরি করুন
অ্যান্ড্রয়েড স্টুডিও ওপেন হওয়ার পর:
1. `Build` মেনুতে যান।
2. `Build Bundle(s) / APK(s)` এ ক্লিক করুন।
3. `Build APK(s)` সিলেক্ট করুন।

কিছুক্ষণ পর আপনার APK তৈরি হয়ে যাবে।

---
### গুরুত্বপূর্ণ: GitHub Actions এর জন্য
আপনি যদি GitHub Actions ব্যবহার করে অটোমেটিক APK বিল্ড করতে চান, তবে নিশ্চিত করুন যে আপনি `android` ফোল্ডারটি আপনার গিট (Git) রিপোজিটরিতে **কমিট (Commit)** করেছেন। 

**কমিট করার কমান্ড:**
```bash
git add android/
git commit -m "Add android platform folder"
git push origin main
```

আপনার GitHub Action-এর `setup-java` স্টেপে `cache: 'gradle'` অপশনটি থাকলে সেটি এরর দিতে পারে যদি রুট ফোল্ডারে গ্রেডেল ফাইল না থাকে। আমি আপনার প্রজেক্টে একটি সঠিক `android-ci.yml` ফাইল তৈরি করে দিয়েছি যা এই সমস্যা সমাধান করবে।

---
**বিঃদ্রঃ:** আপনার ফায়ারবেস (Firebase) কনফিগারেশন অ্যান্ড্রয়েড অ্যাপের জন্য ঠিক করা আছে কিনা তা নিশ্চিত করুন (Firebase console থেকে Android app যুক্ত করুন এবং `google-services.json` ফাইলটি `android/app/` ফোল্ডারে রাখুন)।
